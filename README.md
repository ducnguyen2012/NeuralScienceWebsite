# TO RUN THIS PROJECT, RUN BELOW COMMAND:
## CLONE THE PROJECT:
```bash
git clone https://github.com/ducnguyen2012/NeuralScienceWebsite.git
```
## RUN BACKEND:
```bash
cd NeuralScienceWebsite/backend
```
```python
python server.py
```
## RUN FRONTEND:
```bash
cd NeuralScienceWebsite/frontend
```
```react
npm start
```
